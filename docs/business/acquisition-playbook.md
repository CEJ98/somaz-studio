# Acquisition Playbook

## Objetivo

Construir pipeline calificado para Somaz sin depender de volumen bajo-ticket, algoritmo organico o paid media como motor principal.

## Principio

- pocos leads
- mejor fit
- mayor ticket
- mejor claridad de problema

Somaz vende:

- direccion arquitectonica
- interiores y visualizacion con criterio
- paquetes para permisos, inversores y alineacion
- partner model correcto cuando la jurisdiccion lo exige

No vende:

- renders baratos
- estimados gratis sin brief
- firma local propia fuera del alcance habilitado

## Mix de adquisicion v1

### Prioridad por canal

1. outbound directo B2B
2. partnerships
3. referrals
4. LinkedIn
5. Instagram
6. paid media minimo / retargeting solo si ya existe base

## ICP por canal

### 1. Developers chicos / medianos

Problema a vender:

- falta de claridad para inversores
- presentacion debil para permisos o boards
- demasiadas decisiones abiertas antes de documentacion

Oferta relevante:

- architect-led design direction
- investor / developer deck visuals
- permit-support presentation package

CTA recomendado:

- discovery call
- paid consult si el proyecto es complejo o multijurisdiccional

### 2. Hospitality boutique

Problema a vender:

- interior sin logica comercial clara
- marca y espacio desconectados
- materialidad y experiencia poco cohesionadas

Oferta relevante:

- interior architecture
- concept support
- presentation package for stakeholders

CTA recomendado:

- paid consult
- discovery call si el brief ya esta ordenado

### 3. Architect partners / estudios

Problema a vender:

- presentaciones flojas para cliente
- falta de soporte visual o conceptual
- sobrecarga del estudio en fases tempranas

Oferta relevante:

- visualization support
- architect-led concept support
- client alignment package
- permit presentation material

CTA recomendado:

- discovery call corta
- white-label / referral conversation

### 4. Brokers premium / desarrollo

Problema a vender:

- proyectos/listings mal presentados
- compradores o inversores sin claridad
- developers sin buen material comercial

Oferta relevante:

- listing / development presentation support
- investor visuals
- packaging for confidence and pre-sales

CTA recomendado:

- intro call
- referral / partner conversation

## Cadencia semanal

Capacidad real founder-led: `5-7h`

### Bloque sugerido

- `2h` building list + research
- `2h` outbound personalizado
- `1h` follow-up
- `1h` LinkedIn / Instagram authority
- `1h` partnerships / referrals

## Targets semanales

- `20-30` nuevos contactos outbound bien filtrados
- `5-10` follow-ups
- `2` outreachs de partnership
- `1` post de LinkedIn
- `2-3` piezas de Instagram derivadas del mismo asset
- `1` pedido de referral si hay contexto real

## Targets mensuales

- `80-120` nuevos contactos outbound
- `8-12` conversaciones utiles
- `4-6` discovery calls
- `2-4` paid consults
- `1-2` propuestas serias
- `1` cierre bueno o pipeline suficiente para el siguiente mes

## Reglas por canal

### Outbound

- no vender servicio generico
- abrir con problema visible y contexto real
- no mandar secuencias masivas sin personalizacion minima
- CTA a discovery o paid consult, nunca a “charlar”

### Partnerships

- no salir con mensaje generico de colaboracion
- salir con un offer claro segun tipo de partner
- diferenciar `referral`, `white-label`, `coordinated`

### Referrals

- pedir despues de resultado visible, no antes
- pedir intro concreta, no “si conoces a alguien”
- usar case + one-pager + mensaje corto

### LinkedIn

- autoridad profesional y capa de credibilidad para outbound
- 1 post semanal
- conectar con prospects antes del mensaje cuando tenga sentido

### Instagram

- prueba social + criterio + atmosfera
- menos portfolio mudo
- mas micro-casos con problema, alcance y resultado

### Paid media

- no usar como motor
- solo retargeting o tests chicos cuando:
  - haya landing fuerte
  - tracking listo
  - mensaje probado por outbound

## Mensaje por segmento

### Developer

- clarity before documentation
- investor material
- permit-support package
- faster decisions

### Hospitality

- brand-space alignment
- guest experience
- material system
- repeatable interiors

### Architect partner

- client alignment
- visualization
- concept support
- no competencia por licensure local

### Broker

- stronger project packaging
- more buyer confidence
- clearer development material

### Premium homeowner

- architect-led guidance
- interiors + visualization
- cleaner process

## Paid media

Presupuesto inicial recomendado: `USD 0-300/mes`

Uso permitido:

- retargeting a `/services`, `/services/architecture`, `/services/miami`, `/work`, `/contact`
- tests chicos en Meta o LinkedIn sobre:
  - permit-support presentation package
  - investor-ready visualization
  - architect-led remote design
  - interior architecture for hospitality

No correr:

- awareness generico de arquitectura + interiores
- ads sin landing especifica
- ads sin tracking ni CTA claro

## Source taxonomy

Usar estos valores unicos en CRM:

- `website`
- `instagram`
- `linkedin`
- `email_outbound`
- `whatsapp`
- `referral`
- `partner_referral`
- `broker_referral`
- `paid_social`
- `organic_search`

## KPIs

- source -> reply
- reply -> discovery
- discovery -> paid consult
- paid consult -> proposal
- proposal -> close
- avg ticket por source
- close rate por source
- fit / no-fit por source

## Criterio de exito 90 dias

- pipeline mas ordenado
- source attribution consistente
- 1-2 canales dominantes claros
- referrals activados como sistema
- paid media aun secundaria

Exito real:

- Somaz sabe que canal trae mejor fit
- Sofía puede sostener la cadencia sin romper delivery
- el mensaje comercial atrae menos low-end y mas proyectos con decision real
