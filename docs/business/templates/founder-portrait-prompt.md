# Prompt de retrato para la fundadora

Usar este prompt en GPT Imágenes junto con fotos reales de referencia de Sofía Mazzucco para preservar identidad, rasgos, edad, tono de piel y pelo.

## Prompt maestro

```text
Create a refined editorial portrait of a female architect for the website of a premium architecture studio based in Miami with Latin American roots. Use the provided reference photos only to preserve her real facial features, age range, skin tone, hair, and identity.

Style:
- sophisticated, calm, intelligent, architecture-led
- premium but understated
- not fashion-glam, not influencer, not overly corporate
- natural confidence, direct but warm presence

Composition:
- medium portrait or 3/4 portrait
- eye-level camera
- relaxed posture
- subtle, professional expression
- clean framing suitable for a website About page hero or founder section

Wardrobe:
- minimal, elegant, architectural
- neutral tones such as off-white, stone, black, taupe, soft gray, or muted earth tones
- no loud patterns, no heavy jewelry, no trendy styling

Lighting:
- soft natural light
- high-end editorial realism
- gentle contrast
- skin should look natural, not over-retouched

Background:
- minimalist interior or architectural setting
- warm neutral materials such as plaster, stone, travertine, concrete, wood, linen, or soft shadows
- background should feel expensive and quiet, not busy

Visual direction:
- timeless
- realistic
- subtle luxury
- serious creative professional
- suitable for an architecture studio website

Avoid:
- exaggerated smile
- beauty campaign makeup
- cinematic fantasy look
- overly glossy skin
- fake office stock-photo look
- strong blur, dramatic color grading, or artificial glamour

Output:
- photorealistic
- premium editorial portrait for web
- clean negative space around subject when possible
```

## Variantes

### 1. Vertical para About page

```text
Use a vertical composition, 4:5 ratio, with the subject centered and enough negative space around the upper body for an elegant founder portrait on an architecture studio About page.
```

### 2. Horizontal para hero/editorial

```text
Use a horizontal editorial composition with the architect positioned slightly off-center, leaving calm negative space for website headlines and supporting text.
```

### 3. Retrato neutro secundario

```text
Create a cleaner, more neutral portrait with a softer architectural background and a more direct founder presence, suitable for secondary website sections, press, or social profile use.
```

## Criterios de selección

- Debe leerse primero como arquitecta, no como influencer ni modelo.
- Debe sentirse premium, serena y profesional.
- Debe convivir bien con fondos cálidos, serif editorial y CTAs sobrios.
- Evitar imágenes demasiado oscuras, muy glamorosas o demasiado lifestyle.
