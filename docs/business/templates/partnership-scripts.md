# Partnership Scripts

## 1. Architect-of-record / local architect

### Oferta

- concept support
- visualization for client alignment
- permit / board presentation layer
- packaging previo a submission

### Ask

- referral cuando el cliente necesita mejor frente visual o mas claridad antes de documentacion

### Frase base

Somaz puede reforzar la capa de diseno y presentacion sin competir con tu rol local de licensure, submission o code adaptation.

## 2. Broker

### Oferta

- presentation package para developments
- visuals para listings premium
- material para buyer / investor confidence

### Ask

- intro a developers o owners que necesiten mejor packaging del proyecto

### Frase base

No hacemos marketing generico; ayudamos a que el proyecto se entienda mejor cuando eso impacta confianza, ventas o decision.

## 3. Builder / contractor premium

### Oferta

- interior definition before execution
- material and layout clarity
- client alignment package

### Ask

- intro cuando el cliente llega con brief flojo o demasiadas decisiones abiertas

### Frase base

Somaz ayuda a ordenar la capa de diseno antes de que el proyecto se vuelva mas caro por cambios tardios.

## 4. Hospitality operator / consultant

### Oferta

- interior architecture
- brand-space alignment
- concept and presentation package

### Ask

- intro a proyectos en refresh, expansion o nueva apertura

### Frase base

Si el espacio necesita sostener mejor la marca y la experiencia, ese es el punto donde Somaz entra mejor.
