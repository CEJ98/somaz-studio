¿Cuál es el alcance del update?

El sitio actual usa el sistema oscuro (#1C1C1A). Hay tres niveles de cambio posibles.

A

Solo tokens de color

Cambiar las 5 variables CSS en globals.css y la fuente display de Cormorant → Bodoni Moda. El sitio queda cream/ink con el mismo layout. Rápido, sin riesgo, listo en minutos.

B

Tokens + homepage redesign

Color swap completo + rediseñar las secciones del homepage (Hero, Services, Work, CTA) para que aprovechen el nuevo sistema cream. Las páginas internas se actualizan con los tokens.

C

Redesign completo del sitio

Cada página rediseñada desde cero: homepage, trabajo, servicios, about, blog, contacto. El sistema de identidad V2 aplicado en todo. Mayor impacto, mayor tiempo.
